# DevFreela
*Plataforma para cadastro e contratação de serviços de freelance de desenvolvedores.*

# Tecnologias:
* ASP.NET Core
* .NET 10
* Entity Framework Core - SQL Server
* MediatR
* FluentValidations
* Json Web Token (JWT)
* xUnit + Moq
* Swagger
* SendGrid

# Patterns:
* Rest API
* Clean Architecture
* Clean Code
* Repository
* Command Query Responsibility Segregation (CQRS)

# Funcionalidades:
* Cadastro, atualização, cancelamento e consultas de serviços de freelancer;
* Cadastro de usuários e de perfis freelancer e cliente;
* Adicionar comentários para um serviço de freelancer;
* Definir, iniciar e finalizar projeto.

# Fluxo principal:
* O cliente publica uma oportunidade de projeto na plataforma, com título, descrição, valor e outras informações;
* Um freelancer encontra uma oportunidade e troca mensagens com o cliente;
* Acertadas as condições, o cliente seleciona o freelancer como profissional escolhido e inicia o projeto;
* Vão ser adicionadas mensagens ao projeto, sobre o avanço dele;
* O projeto poderá ser concluído, e o freelancer vai ter seu saldo atualizado, bem como seu perfil.

# Qual o problema:
* Com a alta demanda de novos projetos no mercado, a necessidade de se delegar módulos ou projetos inteiros se torna comum;
* Isso ocorre por diferentes motivos, como por exemplo, prazos apertados, a equipe esta ocupada em projetos internos, ou que conhecimentos especializados estejam ausentes na equipe.

# Solução proposta:
* Plataforma simplificada para o cliente publicar o serviço a ser realizado no seu projeto, conseguindo um alcance muito grande de profissionais que possuem essas expertises;
* Plataforma permite que desenvolvedores encontrem com facilidade projetos que possuam atuar;
* Plataforma que garanta segurança para ambos os lados.
